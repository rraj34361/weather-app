# Weather React App using RapidAPI

This repository contains a Weather Application built with React that utilizes the RapidAPI service to fetch weather details. This application allows users to search for weather information for a specific location and view current weather conditions as well as a 5-day forecast.

## Features

- **Search Weather**: Users can search for weather information by entering the name of a city or location.
- **Current Weather**: Display the current weather conditions including temperature, humidity, wind speed, and weather description.
- **5-Day Forecast**: Show a 5-day weather forecast with details for each day.
- **Responsive Design**: The application is designed to work seamlessly on different devices, including desktops, tablets, and smartphones.

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

- Node.js and npm installed on your system.

### Installation

1. Clone the repository to your local machine:

   ```
   git clone https://github.com/your-username/weather-react-rapidapi.git
   ```

2. Navigate to the project directory:

   ```
   cd weather-react-rapidapi
   ```

3. Install the required dependencies:

   ```
   npm install
   ```

### Configuration

1. Get an API key from [RapidAPI](https://rapidapi.com/).

2. Create a `.env` file in the project root and add your API key:

   ```
   REACT_APP_RAPIDAPI_KEY=your_rapidapi_key_here
   ```

### Usage

1. Start the development server:

   ```
   npm start
   ```

2. Open your browser and navigate to `http://localhost:3000` to see the Weather App in action.
